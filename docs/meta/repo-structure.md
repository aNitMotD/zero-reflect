# Repository Structure

This repository is spec-first. Code is optional.

## Root
- README.md: entry point, non-authoritative overview
- SPEC.md: source of truth for ZR behavior
- LICENSE: AGPL-3.0 full text
- NOTICE: project notice (optional but recommended)

## docs/
- design-constraints.md: minimal invariants (constitution)
- design-notes.md: extended rationale / old notes (non-binding)
- integration-mapping.md: MCP / OpenAI / prompt layer mapping
- test-cases.md: spec-based tests

## src/
- interface.md: language-agnostic reference interface (optional)
